# 每日计划与复盘

一个本地优先的个人桌面应用，用于早上计划、白天执行、晚上复盘。

## 技术栈

- Tauri 2
- React
- TypeScript
- SQLite

## 开发

```powershell
npm install
npm run test:run
npm run build
npm run tauri dev
```

## MVP 范围

- 今日工作台：文件夹视图和星系视图
- 四象限任务管理
- 任务时间记录和手动修正
- 顺延任务确认
- 晚间四问复盘
- 今日完成概览
- 轻量月度总览

## 暂不包含

- 账号和云同步
- 团队协作
- 模板系统
- 自定义原因标签
- 完整月度动画
