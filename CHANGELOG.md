# Changelog

所有值得用户和贡献者知道的变化都会记录在这里。

## 0.2.0 - 2026-06-23

### Added

- 今日工作台：文件夹视图和星系视图。
- 四象限任务管理。
- 飞船任务执行交互：点击开始，再次点击完成并记录实际耗时。
- 未完成任务顺延到明天，并记录顺延原因。
- 晚间四问复盘。
- 今日完成概览。
- 月度总览：完成趋势、重要任务、四象限分布、顺延原因、实际耗时、月度星系地图和月度文件柜。
- 数据安全面板：JSON 备份、JSON 导入、Markdown 档案导出、示例数据重置。
- 本地任务模板。
- 简单重复任务：每天、工作日、每周。
- 本地搜索和筛选。
- Windows GNU 最小桌面工具链脚本。
- 便携 zip 打包脚本。
- Phase 6 开源发布准备：LICENSE、CONTRIBUTING、CHANGELOG、Issue Forms、CI 和 Release 草案。

### Changed

- 计划阶段改为全宽分步任务创建，不再同时展示文件夹或星系视图。
- 执行阶段改为全窗口主视图和按需工具抽屉。
- 星系四象限增加区域色雾、象限同色星球、曲线路径巡航和减少动态效果支持。
- 复盘阶段改为降亮的当日视图背景和右侧分步抽屉。

### Notes

- 当前版本仍是 MVP 便携体验版。
- 默认本地保存 SQLite 数据，不提供账号、云同步或团队协作。
- 便携 zip 不应包含个人 `data/user.sqlite`。
